Introduction

This is a Laravel v10.x application to control notifications using a Dockerized environment.

Base URL: http://localhost:9999

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Notifications

Endpoints for managing notifications

Send a notification

This endpoint is used to send a notification to a specific recipient.

Example request:
curl --request POST \
    "http://localhost:9999/api/notification" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"recipient_id\": 14,
    \"content\": \"xmadotbojxoaubsvaakvwnvodxvphyjvmkfdhakraparbofsiwtgbrfextzkwiqtqmwwfxecn\",
    \"category\": \"professional\"
}"

$client = new \GuzzleHttp\Client();
$response = $client->post(
    'http://localhost:9999/api/notification',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'recipient_id' => 14,
            'content' => 'xmadotbojxoaubsvaakvwnvodxvphyjvmkfdhakraparbofsiwtgbrfextzkwiqtqmwwfxecn',
            'category' => 'professional',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
const url = new URL(
    "http://localhost:9999/api/notification"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "recipient_id": 14,
    "content": "xmadotbojxoaubsvaakvwnvodxvphyjvmkfdhakraparbofsiwtgbrfextzkwiqtqmwwfxecn",
    "category": "professional"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Get all notifications

This endpoint is used to return a paginated list of notifications from the database.

Example request:
curl --request GET \
    --get "http://localhost:9999/api/notification?page=1&page_size=10" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
$client = new \GuzzleHttp\Client();
$response = $client->get(
    'http://localhost:9999/api/notification',
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'query' => [
            'page' => '1',
            'page_size' => '10',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
const url = new URL(
    "http://localhost:9999/api/notification"
);

const params = {
    "page": "1",
    "page_size": "10",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
